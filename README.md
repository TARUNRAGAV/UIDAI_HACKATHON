# Project Overview

This project is part of the UIDAI Hackathon. The primary focus is to develop a user-friendly application that integrates Aadhaar authentication for various services.

## Documentation

### Getting Started

To set up the project locally, please follow the instructions below:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/TARUNRAGAV/UIDAI_HACKATHON.git
   ```

2. **Install Dependencies**
   Navigate to the project directory and install necessary dependencies:
   ```bash
   cd UIDAI_HACKATHON
   npm install
   ```

3. **Run the Application**
   To start the application, use:
   ```bash
   npm start
   ```

### Features

- Aadhaar Integration
- User Authentication
- Easy Access to Government Services
